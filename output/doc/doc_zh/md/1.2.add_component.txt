1.2 在胶片页添加组件
-----------------------

&nbsp;

#### 认识组件与构件

现在我们已打开 `MySlide/index.html` 的编辑页面，呈现在大家面前的界面也称 “可视化设计器”，在它的中心区域是 “主设计区”，被编辑文档的内容在此展示，界面的左侧是 “常用工具栏”，右侧是 “样板资源页”，包括：“属性页”（property）、当前工程资源页（resource）、样板页（slide、Md）等。

> ![Edit project](md/res/creator.png)

在 “主设计区” 用鼠标点击可选中指定构件，选中后，将有针对该构件的浮动工具栏浮于其上，如下图。

> ![Edit project](md/res/popup_btn.png)

点击 Select uplevel 浮动按钮 ![select up](md/res/go_up_btn.png)，可切换到上层构件，即，选中上一层构件。若在当前内容页的空白处点击，胶片页（即 ScenePage，也称场景页）被选中，它是当前页所有内容的顶层容器。

在可视设计器中能被选中的界面元素都称为 **“构件”**，而直接在胶片页下存放的那层构件，专称为 **“组件”**，因为创建新内容，或引用已有内容，通常以 “组件” 为单位，所以，为了讲述方便，直接摆在胶片页下的构件我们称为 “组件”。在主设计区，选中的组件显示蓝色虚线框，而其它构件选中时显示红色虚线框。

#### <a name="rmv_page"></a>删除首页胶片

当前胶片从 `shadow-slide/index.html` 拷贝而来，它的首页用于配置文档类型，我们要到 [第 5 章](#5.) 才介绍，现在不妨先删除这一页。

先在当前内容页的空白处点击，选中页构件后，点击左边栏中 `Remove widget` 按钮 ![remove button](md/res/delete_btn.png) 即删除当前页。然后点击左边栏的 `Save` 按钮 ![save button](md/res/save_btn.png) 存盘。

#### 从资源页拖入样板来创建组件

在胶片页中添加组件，主要有两种方式，其一是从右侧资源页中，用鼠标拖一个样板进来，比如，我们现在切换到 `slide` 资源页，把标为 `P` 的样板拖到主设计区的空白区域，松开鼠标后一个段落组件立即在当前位置创建。其二是通过复制方式创建内容组件，操作过程随后介绍。

![slide resource](md/res/slide_res.png)

拖入样板创建的构件，还可插入到当前已有组件的内部，视我们拖到哪个位置松开鼠标而定，一般情况下，创建内容组件，应将样板拖到当前页的空白位置。如果把拖入的样板扔在某构件上，新建的构件将插在目标位置之前创建，如果扔在这个标记 ![insert flag](md/res/insert_btn.png) 上，表示在目标构件的下一层创建，新构件添加到下层构件的尾部。

#### 选取组件

在当前胶片页，可用 "shift + up" 或 "shift + down" 热键依次切换当前选中的首层构件。按 "shift + right" 则进入当前选中构件的子层（选首个子构件），按 "shift + left" 则进入父层。

如果想选择多个内容组件，可以用按下鼠标左键，向左上方或右下方移动一段距离后松开，即，通过拖移一个区域来选择组件。**如果向左上方拖移，凡被拖移区域包含或与之相交的组件均被选中，如果向右下方拖移，只有被拖移区域包含的组件被选中。**

当多个内容组件被选中时，还可以按住 Shift 键，再用鼠标点选，来添加或减少选中的组件。

选中单个或多个组件后，您可以：
1. 用 `Cmd + C` 或 `Ctrl + C` 热键拷贝，拷贝后可用 `Cmd + V` 或 `Ctrl + V` 粘贴   
注意，当被粘贴的组件将在原位置创建，两个同样的构件叠加在起，要挪走上面新创建的，才能找到下面旧有组件。
2. 用鼠标移动组件
3. 点击 ![remove](md/res/delete_btn.png) 按钮删除选中的组件
4. 用 "alt + del" 热键删除选中的组件   
这种删除方式不弹出对话框确认，但上一种删除方式要确认。
5. 用 "alt + left, alt + top, alt + right, alt + bottom" 热键移动选中的组件   
每按一次热键，移动一个像素，此操作常用于精调组件位置。

说明：
1. 用热键拷贝构件，需在构件选中后立即进行，否则，当前焦点可能被迁移，导致无法用热键拷贝选中的构件。
2. 对于 Mac 桌面系统，用 `Cmd + C` 拷贝（即：按下 Command 键后再按 C 键），其它桌面系统用 `Ctrl + C` 热键，其它热键如 `Cmd + V, Cmd + X` 等，情况类似。

#### <a name="copy"></a>用浮动按钮复制

上述用热键 "Cmd + C" （或 "Ctrl + C"）拷贝，用 "Cmd + V"（或 "Ctrl + V"）粘贴进行复制非常有用，当别人制作各种界面组件，成为样例，保存在胶片文档里，你只需打开文档，用拷贝粘贴即实现了共享与复用。

此外，用户还可先选中构件，然后拖动拷贝浮动按钮 ![copy](md/res/copy_btn.png)  到目标位置实现复制。举个例子，我们从右侧 slide 资源页拖入一个 `Ul` 样板，在当前页中创建一个组件后，选中列表第一项，然后拖动浮动按钮 ![copy](md/res/copy_btn.png) ，扔到当前选中的自身构件上。

![copy](md/res/copy_by_btn.png)

您将得到如下图拷贝结果：

![copy](md/res/copy_result.png)

#### 取消选中

如果想取消已选中的一个或多个组件，只须用鼠标点击标尺，点击上标尺或左标尺均可。
